Documentation Page

This movie API will allow users to sign up, create a list of their favourite movies, change their personal information, as well as accessing general information of different movies.

Business Logic Endpoint URL Method Type Query Parameters Body of the Request Format of the response data
Creates a User /users POST None A JSON object of the new user, structured like: {"Username": "", "Password": "", "Email": "", "Birthday": "0000-00-00"} A JSON object of the new user, structured like: {"Username": "", "Password": "", "Email": "", "Birthday": "0000-00-00"}
Returns list of all movies /movies GET None None A JSON object of all the movies
Returns data about single movie by title /movies/:Title GET :Title None A JSON object a specific movie
Returns data about a genre by name /movies/genres/:Name GET :Name None A JSON object of a specific genre
Returns data about a director by name /movies/directors/:Name GET :Name None A JSON object of a specific director
Returns a list of all users /users GET None None A JSON object of all the users
Returns a list of a single user by username /users/:Username GET :Username None A JSON object of the specfic user
Allows users to update their information /users/:Username PUT :id A JSON object of the new user, structured like: {"Username": "", "Password": "", "Email": "", "Birthday": "0000-00-00"} A JSON object of the new user, structured like: {"Username": "", "Password": "", "Email": "", "Birthday": "0000-00-00"}
Allows users to add a movie to their favorites /users/:Username/movies/:MovieID POST :Username, :MovieID None A JSON of the user
Allows users to remove a movie from their favorites /users/:Username/movies/:MovieID DELETE :Username, :MovieID None Text confirming the movie has been removed from the user's list
Allows a user to be removed /users/:Username DELETE :Username None Text confirming the user has been deleted
